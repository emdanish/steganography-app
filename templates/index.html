What is Steganography?

Steganography is the practice of hiding secret information within ordinary, non-secret data or a physical object. In this application, you can hide an entire image inside another image.

How to Use

To Hide: Select a carrier image (the visible container), a secret image to hide, and click "Hide Image".

To Extract: Select a steganographic image created by this tool and click "Extract Image".

Security Note

For best results, the carrier image should be larger than the secret image. The extraction quality depends on the image characteristics.
